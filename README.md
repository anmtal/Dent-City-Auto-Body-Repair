# Dent City Auto Body Repair — Website

A 5-page website for **Dent City Auto Body Repair**, an auto **body / collision** shop in Burlington, ON. Built by **Apex Web Studio** as a free sample.

- **Live:** https://anmtal.github.io/Dent-City-Auto-Body-Repair/
- **Instagram:** https://www.instagram.com/dentcityautobodyrepair/

## Pages
- `index.html` — hero, services, how-it-works, about, trust stats, work gallery + lightbox, reviews, hours & location.
- `services.html` — full price guide (collision repair, paintless dent repair, auto painting & refinishing, frame & structural, bumper & scratch, detailing) + FAQ.
- `diagnostics.html` — **Insurance Claims** page: how we handle your claim start-to-finish, direct billing, process steps.
- `book.html` — multi-step free-estimate request form with live estimate handoff, FAQ, contact.
- `contact.html` — contact cards, hours & location (NAP), directions.

## Stack
Static HTML/CSS/JS — no build. **Dark theme + signal-red** design system (via the ui-ux-pro-max skill: surface #0B0E15 / #161B26, primary red #DC2626, amber #F59E0B accent), Oswald + Inter. Animation is CSS-driven (scroll reveals, marquee, ambient hero blobs/gears) in `css/styles.css` + interactions in `js/main.js` — no external motion library.
